Código para rodar um museu interativo

Material criado em trabalho interdisciplinar na faculdade, com o intuito de criar um aplicativo simples em C rodando no console direto, onde fosse possível o cliente comprar um ou mais ingressos para 4 exposições diferentes, tendo a possibilidade de escolher o tipo de ingresso (inteira, meia ou isenção de entrada) e em outra máquina ter cada uma das exposições escolhidas com no final um breve questionário sobre a exposição visitada, a fim de avaliar se o conteúdo apresentado foi fixado para cada visitante

Foi escolhido usar uma máquina para a venda de ingressos e mais uma máquina para cada exposição, totalizando 5 computadores diferentes, por isso 5 arquivos de código em C
Meu primeiro trabalho no meio, mas não devemos nos envergonhar de nossos começos..
